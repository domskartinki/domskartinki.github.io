Privacy Policy

Dom s kartinki Pin Scheduler is a private internal tool used only by the owner of the Pinterest account.

The tool helps prepare, approve, schedule, and publish organic Pins to the owner’s own Pinterest boards.

The application does not collect, sell, share, or store personal data from other Pinterest users.

The application may use Pinterest account data only to:

  • read the owner’s Pinterest boards;
  • create and publish Pins approved by the owner;
  • manage the publishing queue for the owner’s own content.

No third-party user data is collected or processed.

If Pinterest access is disconnected, the application will stop accessing Pinterest data.

This application is not affiliated with, endorsed by, or sponsored by Pinterest.
